# Vision

LinguaFlow is a modern Windows desktop application that allows users to write naturally in English while an AI simultaneously produces a polished document in another language (initially Spanish).

This is **NOT** a traditional translator.

Instead, the AI rewrites the document as if it were originally written by a native speaker while preserving:

- Meaning
- Tone
- Formatting
- Paragraphs
- Lists
- Headings
- Professional structure

The goal is to feel like writing in Microsoft Word while a professional bilingual writer creates the translated document in real time.

---

# Example

## User Types

> Hello,
>
> I would like to schedule a meeting next Tuesday.
>
> Thank you.

## AI Produces

> Hola,
>
> Me gustaría programar una reunión para el próximo martes.
>
> Muchas gracias.

Notice that this is not a literal translation.

The AI rewrites the text naturally.

# Core Features

## Live Translation

When the user pauses typing (approximately 400 milliseconds):

1. Detect changed paragraph
2. Send only that paragraph to Ollama
3. Receive translation
4. Replace translated paragraph
5. Preserve remaining document

Never regenerate the entire document.
